Etiquetas Semánticas Estructurales

Div es una Etiqueta contenedora que no tiene semantica
Header es la cabecera de un sitio web o una sección
Main Define la seccion principal del documento, sólo puede existir una etiqueta main por documento
Footer- Pié de pagina de un sitio web o de una sección
nav- Representa una navegación
Article- Representa una sección autocontenido(que por si sola se explica)
aside- Representa contenido complemntario o secundario
section- Representa una sección de contenido genérico
address-Representa una direción u ubicación e información de contacto

Etiquetas de Bloque VS Etiquetas de Linea

La etiqueta de bloque es igual a = DIV, estas hacen referencia hacen que sus etiquets hermanas se genere un salto de linea
La etiqueta de Linea es igual a = span, estas solo ocupan el espacio necesario y no generan salto de linea con sus etiquetas hermanas
